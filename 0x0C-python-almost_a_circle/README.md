+ Import
+ Exceptions
+ Class
+ Private attribute
+ Getter/Setter
+ Class method
+ Static method
+ Inheritance
+ Unittest
+ Read/Write file
+ args and kwargs
+ Serialization/Deserialization
+ JSON
